# GEO-Optimized Blog Post Template for camb.ai

Use this template for every new blog post. It incorporates all Princeton GEO methods for maximum AI citation visibility.

---

## Pre-Writing Checklist

- [ ] Target keyword identified (primary + 2-3 secondary)
- [ ] Search intent understood (informational, commercial, transactional)
- [ ] 3-5 authoritative sources gathered for citations
- [ ] 5-10 specific statistics/data points collected
- [ ] 1-2 expert quotes secured (internal team or external)
- [ ] Competitor content analyzed (what exists, what's missing)
- [ ] FAQ questions drafted (3-5, based on "People Also Ask" + search autocomplete)

## Post-Writing Checklist (GEO Quality Score)

### Princeton GEO Methods Applied:
- [ ] **Citations**: 3-5 authoritative sources cited with links (+40% visibility)
- [ ] **Statistics**: Specific data points every 150-200 words (+37%)
- [ ] **Quotations**: Named expert quote(s) included (+30%)
- [ ] **Authoritative Tone**: Confident, definitive language throughout (+25%)
- [ ] **Clarity**: Short paragraphs (< 120 words), plain language (+20%)
- [ ] **Technical Terms**: Domain-specific terminology included naturally (+18%)
- [ ] **Unique Words**: Varied vocabulary, no repetitive phrasing (+15%)
- [ ] **Fluency**: Smooth transitions, clear structure, high readability (+15-30%)
- [ ] **No Keyword Stuffing**: Primary keyword appears 3-5x naturally (not forced)

### Structural Elements:
- [ ] TL;DR box at top
- [ ] Direct answer in first 40-60 words
- [ ] Clear H1 > H2 > H3 hierarchy
- [ ] At least 1 comparison table or structured data element
- [ ] Numbered lists or bullet points in every major section
- [ ] FAQ section with 3-5 questions
- [ ] Internal links to 3-5 related pages
- [ ] CTA to product/demo page
- [ ] Word count: 2,000+ words

### SEO Checklist:
- [ ] Unique title tag (50-60 chars) with primary keyword
- [ ] Unique meta description (150-160 chars) with keyword + CTA
- [ ] Self-referencing canonical URL
- [ ] OG + Twitter Card tags set
- [ ] Featured image with descriptive alt text
- [ ] All images have alt text
- [ ] BlogPosting + FAQPage JSON-LD schema added
- [ ] Published date and author byline visible
- [ ] Category/tag assigned
